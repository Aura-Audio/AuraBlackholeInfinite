Blackhole / Infinite Reverb

Live microphone processing into an extremely long, modulated, pitch-shifted reverb cloud. The design uses a feedback-delay network with high feedback, damping filters, modulation, and a granular pitch-shifting pre-stage.

Use headphones. Loudspeaker monitoring may cause severe acoustic feedback. A secure context such as localhost or HTTPS is required for microphone access.

Parameters

Presets

Decay at 100% forces infinite feedback. Freeze mutes the input and sustains the existing cloud. Clear Reverb Buffer resets the delay network if low-frequency energy accumulates.
